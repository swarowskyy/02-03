Regras para poder adquirir: Promoção "Código Premiado"

1. Formato: O código deve ser digitado exatamente como aparece (os treze dígitos) para adquirir a promoção.
2. Legibilidade: Números ilegíveis tornam o código INVÁLIDO.
3. Limite: Máximo de 2 códigos por CPF/dia.
4. Resgate: Prazo de 5 dias após a validação para retirar o seu querido brinde.
5. Comprovação: Guarde o recorte original; podemos solicitar foto.
